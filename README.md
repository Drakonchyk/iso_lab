# Lab work: Creating ISO image
Authors (team): [Anastasiia Pelekh](https://github.com/Drakonchyk)<br> 

## Prerequisites

- **c++**
- **mkisofs**

### Installation

```bash
git clone https://github.com/Drakonchyk/iso_lab
```

### Usage
```bash
mkdir build
cd build
cmake ..
make
./bin/create_iso <path_to_directory> <path_to_iso>
```
